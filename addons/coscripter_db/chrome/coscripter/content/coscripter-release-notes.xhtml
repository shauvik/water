Release Notes for M9

Milestone 9 (version 1.901) was released on June 4th, 2010.

Support for more DOJO-based widgets

Support for JavaScript expressions in steps

You can use simple JavaScript expressions to create more flexible strings for entering values. 

enter j"1+2+3" into the "hello world" textbox

will enter 6 into the hello world textbox.

Slightly more cryptic but also more useful:

enter j"new Date(new Date().getTime()+7*24*60*60*1000).toLocaleDateString()" into the "Date" textbox

will enter next week's date into the "Date" textbox
